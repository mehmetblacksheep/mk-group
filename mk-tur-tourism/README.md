# MK Tur Tourism

Bu paket kullanıcı tarafı çalışan frontend prototipidir.

## Çalıştırma

```bash
npm run dev
```

Ardından tarayıcıda aç:

```text
http://localhost:3000
```

## İçerik

- `index.html`: ana kullanıcı ekranı
- `styles.css`: stil dosyası
- `app.js`: kategori, liste ve popup davranışları
- `admin/index.html`: admin panel iskeleti
- `assets/logo.png`: yüklenen logo

## Not

Bu sürümde başlangıçta hiçbir kategori seçili değildir. Önce 3 büyük seçenek görünür.
